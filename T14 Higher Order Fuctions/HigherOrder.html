Array Foreach

Foreach takes each element of array and pass them as argument to callback functions. It does't return new array it alter the existing one.

Here adding 10 to each element using foreach

Array Map

Map function takes each element of array and pass them as argument to callback functions and perform operations on them and returns new array.

Here adding 10 to each element using Map function

Array Filter

Filter function takes each element of array and only returns those elements in new array who satisfies the provided condition

Here fitering elements which are grater than 50

Array Reduce

Reduce function takes each element and perform operations with the output of previous iteration and generates a single valued output.

Here addition of all elements

Array Sort

Sort function sorts the values of array in descending order we can also pass the callback function to perform better sorting. It alters the original array values
